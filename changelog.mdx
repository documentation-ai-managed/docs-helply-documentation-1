## Upgrade Your Integration

Follow these steps to update your Helply integration to the latest version.

<Steps>
  <Step title="Update SDK" icon="download">
    Install the latest package based on your platform.

    <CodeGroup tabs="Node.js,Python">
      ```bash
      npm install @helply/sdk@latest
      ```
      ```bash
      pip install helply-sdk==1.2.0
      ```
    </CodeGroup>
  </Step>
  <Step title="Migrate API Calls" icon="settings">
    Update endpoint paths from `/v0/tickets` to `/v1/tickets`.

    <CodeGroup tabs="JavaScript,Python">
      ```javascript
      const response = await fetch('https://api.helply.ai/v1/tickets', {
        method: 'POST',
        headers: { 'Authorization': `Bearer ${API_KEY}` },
        body: JSON.stringify({ content: 'Customer query' })
      });
      ```
      ```python
      import requests
      response = requests.post(
          'https://api.helply.ai/v1/tickets',
          headers={'Authorization': f'Bearer {API_KEY}'},
          json={'content': 'Customer query'}
      )
      ```
    </CodeGroup>
  </Step>
  <Step title="Test and Deploy" icon="rocket">
    Run your test suite and monitor the first 100 tickets post-deployment.
  </Step>
</Steps>

## Check Versions Across Platforms

Verify your setup matches the latest releases.

<Tabs>
  <Tab title="Web Dashboard" icon="monitor">
    Navigate to Settings > About to view the deployed version.
  </Tab>
  <Tab title="API" icon="api">
    Call `GET /v1/status` to retrieve server version.

    ```javascript
    const status = await fetch('https://api.helply.ai/v1/status');
    console.log(await status.json()); // { version: "1.2.0" }
    ```
  </Tab>
  <Tab title="SDK" icon="package">
    Check your `package.json` or `requirements.txt` for the installed version.
  </Tab>
</Tabs>
